# May（五月）v0.1

一门自研脚本语言，单文件即可运行。

## 使用方法

### 1. 克隆仓库
bash
git clone https://github.com/10yuki-GH/may-lang.git
cd may-lang
### 2. 交互模式
bash
python3 may.py
### 3. 执行文件
bash
python3 may.py hello.m
## 示例（hello.m）
text
output("May 正在执行文件")
## 语言特性
- 函数：fun / return / noreturn
- 类型：str / int
- 控制：bool / True / False / while
- 输入：input
- 输出：output
- 变量：as / newvar
- 混合列表
- 数字自动转为字符串

## 注意事项
- macOS / Linux 使用 python3
- Windows 使用 python
- .m 文件必须与 may.py 同目录
